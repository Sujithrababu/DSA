# 🚀 2027 — DSA Problem Tracker

## 📌 Overview

This project is a **DSA (Data Structures & Algorithms) Problem Tracker** designed to help students systematically practice and track important coding problems.

The application provides a curated list of **172 high-quality problems** across multiple platforms, categorized by difficulty and frequently asked in top companies.

It is built as a **simple, interactive HTML-based dashboard** that allows users to track progress, filter problems, and stay consistent in preparation.

---

## 🎯 Purpose

The main goal of this project is to:

* Practice **important DSA problems**
* Track solved problems efficiently
* Prepare for **coding interviews and placements**
* Focus on **company-wise frequently asked questions**

---

## 📊 Features

### ✅ Problem Tracking

* Mark problems as completed using checkboxes
* Progress is saved using browser local storage
* Visual completion indicators (strike-through + highlight)

### 📈 Progress Monitoring

* Total problems solved count
* Difficulty-wise tracking:

  * Easy
  * Medium
  * Hard
* Overall progress percentage with progress bar

### 🔍 Smart Filtering & Search

* Filter by:

  * Difficulty (Easy / Medium / Hard)
  * Completed problems
* Search problems by:

  * Name
  * Company tags
* Platform-based filtering

### 🌐 Platforms Covered

* LeetCode

* Codeforces
* HackerEarth
